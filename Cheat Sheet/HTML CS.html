HTML Cheat Sheet

Table Tags

Tag Name Description
< table > Table This element used to create a table.
< thead > Table Head Represents the header section of a table, containing column labels or headings.
< tbody > Table Body Element used to group and define the main content of a table, containing rows of data within a table structure.
< tr > Table Row Represents a table row within an HTML table structure.
< td > Table Data Represents a table data cell used to hold and display content within a table row.
< tfoot > Table Foot Represents the footer section of an HTML table, typically used for summaries or annotations.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns the cell should span horizontally.
rowspan Row Span Specifies how many rows a cell should span vertically within a table.
